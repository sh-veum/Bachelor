# Urls

### Backend

Backend REST Api Swagger docs:

```
http://localhost:8088/swagger
```

Backend GraphQL:

```
http://localhost:8088/graphql
```

```
http://localhost:8089/startSensor

```
http://localhost:8089/stopSensor
